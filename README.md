查看Django版本

```
>> python
>> import Django
>> django.get_version()
```

初始化环境

```
cd typeidea 
pip3 install -r requirements
```


创建项目

`
django-admin startproject typeidea
`

项目完整结构
```
.
├── CHANGELOG.md
├── LICENSE
├── README.md
├── requirements.txt
└── typeidea
    ├── manage.py
    └── typeidea
        ├── __init__.py
        ├── settings.py
        ├── urls.py
        └── wsgi.py
```
